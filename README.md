# student-record-app
A web app that reads and displays student records from a CSV file.

## Features
- Upload any CSV file with student records
- Search students by name, father name, and age
- Filter by grades using dropdowns
- Sort by any column
- Highlights A+ grades in green and F grades in red

## How to use
1. Open index.html in a browser
2. Click "Choose File" and upload the Students.csv file
3. Use the search boxes and dropdowns to filter students
4. Click any column header to sort

## Technologies used
- HTML
- CSS
- JavaScript
